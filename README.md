# Blog Platform

A feature-rich blog platform where users can create, read, update, and delete articles. Built using Spring Boot and Thymeleaf.

## Features

- User Authentication and Authorization
- CRUD operations for blog posts
- Commenting on posts

## Prerequisites

- Java 8 or higher
- Maven
- MySQL Database
